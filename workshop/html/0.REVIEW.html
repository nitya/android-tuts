Topics

The following topics are on the finals:

  • Android Architecture
  • Application Components
  • Android Resources, Manifest, Permissions
  • Testing, Emulation & Tools (ADT, ADB, AVD, DDMS,..)
  • Activities & Fragments
  • Intents, Intent Filters & Intent Resolution
  • Content Providers & Resolvers
  • User Interface (Layouts, Widgets, Views, View Groups) & Action Bar
  • Lists, ListView and ListAdapter
  • Processes & Threads (Async Tasks, UI Thread)
  • Data & Storage (Content Providers, Resolvers, Shared Preferences, SQLLite)
  • Services & Broadcast Receivers (basic knowledge)
  • Camera & Location (basic knowledge)

Lectures

Series 1 and Series 2 link to class lectures. Series 2 leans towards flipped classroom with some concepts covered in class, and others cited for self-study. These will be useful in answering the first-half of the exam (multiple-choice, short-answers, _based on theory_)

Series 1 (Lectures, Google Drive)

  1. Android Quick Tour ==> What is Android, Android Architecture, Android Security Model, Android Versions, Android Application Components
  2. Getting Started ==> Application Components, Android ADT, Application Workflow, Basic App (Activity, Manifest)
  3. The Android Application ==> Key concepts, App Widgets, Components overview (Activity, ContentProvider, Broadcast Receiver, Service), Intents, Intent Filters, Intent Matching
  4. Testing And Debugging I ==> Android Emulator, Android Virtual Device, DDMS, LogCat
  5. Testing And Debugging II ==> AVD Creation, Android Debug Bridge (ADB)
  6. Android Resources ==> XML Review, Android Manifest, Android Resources, Resource Types (drawable, layout, menu, string, style), Resource configurations (res/* directory), Qualifiers, Accessing Resources (Java vs. XML), Usage (adaptive layout, localization)
  7. Android Project Analyze app, CW tutorials walkthrough, Android vs. App widgets
  8. Android User Interface Widgets vs. Containers, Views & ViewGroups, Layouts, Attributes
  9. Android Activity Activity lifecycle, Tasks, BackStack

Series 2 (Lectures, GitHub Slides)

  1. Android Slides Intro Top level topics in Android
  2. Getting Started With Development Building the first app, Adding Action Bar, Supporting Different Devices, Activity Lifecycle
  3. Building a Dynamic UI with Fragments Fragment lifecycle, Fragment Types, Adding Fragments Programmatically/Declaratively
  4. Content Sharing Content Providers and Resolvers
  5. Interaction and Engagement Best Practices for Designing your Application UI & Flow
  6. Services and Background Jobs Service Definition, Types (Bound, Unbound), Lifecycle
